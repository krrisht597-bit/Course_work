**Class and Object:-

1.Class: A class is a blueprint for creating objects (a particular data structure), defining
initial states (attributes), and behaviors (methods).

2.Object: An object is an instance of a class. It is the actual entity created based on the
class blueprint

**Concepts in Python Classes:-

1.Class Definition: A class is defined using the class keyword, followed by the class name
(in CamelCase style by convention).

2.Attributes: Attributes are variables that hold data for the objects created from the class.
These can be instance attributes (specific to an object) or class attributes (shared by all
instances of the class).

3.Methods: Methods are functions that define the behaviors of the class. They typically
operate on the attributes of the object or class and can be used to modify or interact with
the object's data.

4.Constructor (__init__): The constructor is a special method that is called when an object
is created from the class. It initializes the object's state (attributes).

**Instance vs. Class Attributes:-
1.Instance attributes are specific to each object created from the class.

2.Class attributes are shared by all instances of the class.

1.Instance Methods: Methods that operate on instances of the class, using the instance’s
attributes.

2.Class Methods: Methods that operate on the class itself rather than individual instances.
These methods are defined with the @classmethod decorator.

3.Static Methods: Methods that do not operate on an instance or the class itself but behave
like normal functions. These methods are defined with the @staticmethod decorator

Inheritance → Ek class dusri class ke features use karti hai.
Parent Class → Jis class se features liye jaate hain.
Child Class → Jo class features inherit karti hai.
Method Overriding → Child class parent ke method ko change kar sakti hai.

Types of Inheritance (Easy Notes)
Single Inheritance → Ek child class, ek parent class se inherit karti hai.
Multiple Inheritance → Ek child class, ek se zyada parent classes se inherit karti hai.
Multilevel Inheritance → Ek class dusri class se inherit karti hai, aur woh class kisi aur class se inherit karti hai.
Hierarchical Inheritance → Ek parent class se multiple child classes inherit karti hain.
Hybrid Inheritance → Different inheritance types ka combination.

**access modifiers:-

In Python, access modifiers are used to limit access to class variables and methods. Unlike languages like Java or C++, Python does not strictly enforce these restrictions. Instead, it relies on naming conventions to indicate how members should be used.

name mangling:-
In Python, name mangling is a mechanism that automatically renames any identifier within a class that starts with at least two underscores (e.g., __attribute) and ends with at most one underscore.

Encapsulation Python mein ek OOP concept hai jisme data (variables) aur methods (functions) ko ek class ke andar bundle kiya jata hai. Iska main purpose object ki important details ko hide karna aur sirf zaroori cheezein access karne dena hota hai.
