BODY_PARTS = 3 → Starting mein snake ke 3 body parts honge.

7. next_turn()
def next_turn(snake, food):
Game ka main loop hai.
Har 100 ms baad run hota hai.

Snake Head Position
x, y = snake.coordinates[0]
Head ki position lete hain.

Direction Check
if direction == "up":
    y -= SPACE_SIZE
Up → y kam.

elif direction == "down":
    y += SPACE_SIZE
Down → y badhega.

elif direction == "left":
    x -= SPACE_SIZE
Left → x kam.

elif direction == "right":
    x += SPACE_SIZE
Right → x badhega.

New Head Add
snake.coordinates.insert(0,(x,y))
New head list ke start mein add hota hai.

Example:
Before:
[(20,20),(20,40),(20,60)]

After move
[(40,20),(20,20),(20,40),(20,60)]
Head Draw
square = canvas.create_rectangle(...)
Naya block draw hota hai.

8. Food Eat Check
if x == food.coordinates[0] and y == food.coordinates[1]:
Agar snake food tak pahunch gaya.

Score Increase
score += 1
Score +1.

Score Update
label.config(text=f"Score: {score}")
Label update hoga.

Old Food Delete
canvas.delete("food")
Purna food remove.

New Food Generate
food = Food()
Naya food aa jayega.


9. Agar Food Nahi Khaya
else:

Tail remove hogi.
Last Coordinate Delete
del snake.coordinates[-1]
Last body part remove.

Last Rectangle Delete
canvas.delete(snake.squares[-1])

Canvas se bhi delete.
Isse snake move karta hua lagta hai.

12. check_collisions()
def check_collisions(snake):
Collision check karta hai.

Wall Collision
if x < 0 or x >= WIDTH:
Left ya right wall.

if y < 0 or y >= HEIGHT:
Top ya bottom wall.

Body Collision
for body_part in snake.coordinates[1:]:
Head ko body ke saath compare karta hai.

if x == body_part[0] and y == body_part[1]:
Body se takra gaya.
Return:
True
